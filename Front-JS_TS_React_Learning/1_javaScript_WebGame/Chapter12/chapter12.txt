# 지뢰찾기 게임

1. 동작
  - 테이블 생성 및 지뢰 랜덤하게 심기
    (대기)
    (칸 좌클릭 시)
  - 물음표 또는 깃발이 있으면 대기,
  - 폭탄이라면 터지고 게임 종료
  - 폭탄이 아니라면 카을 열고 주변 칸의 개수를 표시
    - 주변 칸 수가 0이라면 옆 칸을 열 수 있으면 연다.
  - 모든 칸을 다 열었으면 이겼다고 표시 및 종료
    (칸 우클릭 시)
  - 물음표 칸으로 전환
  - 물음표 칸 이라면 깃발로 전환
  - 깃발 칸이라면 닫힌 칸으로 전환

2. self-check
  - 첫 번째 클릭한 칸은 지뢰가 없게 만들기
