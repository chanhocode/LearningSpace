# 텍스트 RPG RPG 게임

1. 동작
  - 모드
    1) 모험, 휴식, 종류 중에서 선택하는 일반 모드
    2) 모험을 떠나서 적을 만나게 될 때 돌입하는 전투 모드
  - 전투모드에서는 적을 공격하거나 체력을 회복하거나 도망간다.

2. self-check
  - 휴식, 종료, 회복, 도망 기능 구현

3. 지식
  - 배열 참조, 얕은 복사 깊은 복사
    1) 참조 arr2 = arr1
    2) 얕은 복사 arr2 = (...arr1)
    3) 깊은 복사 arr2 = JSON.parse(JSON.stringify(arr))
  - this : 기본적으로 window를 가르키지만, 객체안에서는 해당 객체를 가르킨다.
    1) 화살표 함수 일때 에는 this가 window가 된다.
  - 클래스 : 객체를 생성하기 위한 템플릿(서식)이다.
