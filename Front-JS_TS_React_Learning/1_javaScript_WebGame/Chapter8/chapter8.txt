# 반응속도 테스트 _ 반응 속도가 얼마나 빠른지 확인하는 프로그램
1. 동작
  - 처음화면 파랑 :: 대기화면
  - 화면을 클릭하면 빨강화면 전환 :: 준비화면
  - 임의의 시간이 지난 후 초록화면으로 자동 전환
  - 초록화면으로 바뀐 뒤 클릭한 시간을 측정 하여 반환 (초록화면이 뜬 시간과 클릭한 시간의 차)
  - 만약 빨강화면에서 클릭 시 성급했다는 메시지 나오고 대기화면으로 전환

2. self-check
  - 가장 빠른 다섯 번의 시도가 몇초 인지를 보여라 ( 6번 이상의 시도한 경우 상위 5개만 보여준다.)

3. 지식
  - 태그.classList.contains('클래스') : 해당 클래스가 들어있다면 true, 업다면 false 반환
  - 태그.classList.add('클래스') : 클래스 추가
  - 태그.classList.replace('기존 클래스', '수정 클래스') : 수정
  - 태그.classLIst.remove('클래스') : 제거

  - reduce.(누적값, 현재값, 인덱스) => (동작 _ 다음 누적 값), 초기값) // 초기값을 넣지 않으면 첫번째 값이 된다.
