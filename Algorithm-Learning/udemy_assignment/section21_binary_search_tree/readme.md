## Trees

- 노드로 이루어진 데이터 구조
- 노드들 사이에 부모-자식 관계가 있다.
- 비선형구조
- 관련 용어
  - Root: 트리 꼭대기 노드 (최상위 노드)
  - Child: 루트에서 멀어지는 방향으로 연결된 노드 (하위 노드)
  - Parent: 자식의 반대 (상위 노드)
  - Siblings: 같은 부모를 가지는 노드
  - Leaf: 자식이 없는 노드 (단말 노드)
  - Edge: 간선 \_ 한노드에서 다른 노드로 향하는 화살표

## Kinds of Trees

- Trees
- Binary Trees
- Binary Search Trees
